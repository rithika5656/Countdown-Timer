A simple and elegant countdown timer web application that allows you to set a target date and displays the remaining time in days, hours, minutes, and seconds.

## How to Use

1. Open `index.html` in your web browser
2. Enter an event name (optional)
3. Select a target date and time
4. Click "Start Countdown"
5. Watch the countdown in real-time!

## Technologies Used

- HTML5
- CSS3 (with Flexbox and CSS Grid)
- Vanilla JavaScript
- localStorage API
